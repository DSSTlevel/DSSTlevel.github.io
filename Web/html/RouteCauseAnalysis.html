Route Cause Analysis - The process of identifying the original fauilt through analysising factors that had followed on from the original issue
Fish Bone Diagram - Visual understanding of the root cause analysis route. Assorting ideas into categories

How to create a Fishbone Diagram?
Agree on a problem statement
Idenfity the main categories to be inserted into the diagram
Write the main categories to be decomposed down into small problems
Identify all the possibilities that had originally caused the issue.
Final analysis of the problem to see if the search matches

Example
An example of this would be to understand the original cause of a data leakage found from a database through identifying an alternative solution and decomposing it down to see if the route matches the issue.
