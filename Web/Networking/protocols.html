IP Packets

    Version - Current version of the IP currently being utilised
    IHL - Header Length in 4-byte units
    QOS - Quality of Service, makes important packets the priority during transmission
    Length - Length of the packet across the network
    ID - The packet number
    Fragment Offset - Packet is broken down (Fragmented), then it is delivered and reassembled
    TTL - Time to live, manages the caching and boosts performance
    Protocol - The data in the protocol that the internet uses
    Checksum - Detects corruption in the header
    Source IP - The starting point of the IP packet
    Destination IP - Its targetted end point
